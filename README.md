## 📝 Blog Posts

### blog-1.md — `any` vs `unknown` এবং Type Narrowing

- `any` কেন "type safety hole" বলা হয়
- `unknown` কেন safer choice
- Type Narrowing-এর বিভিন্ন পদ্ধতি: `typeof`, `instanceof`, `in`, Custom Type Guard

### blog-2.md — OOP-এর চারটি স্তম্ভ

- **Encapsulation** — `private` দিয়ে data সুরক্ষিত রাখা
- **Inheritance** — Parent class থেকে feature পাওয়া
- **Polymorphism** — একই method, ভিন্ন আচরণ
- **Abstraction** — জটিলতা লুকিয়ে সহজ interface দেওয়া
